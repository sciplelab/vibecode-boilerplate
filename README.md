# vibecode-boilerplate

Boilerplate/template for vibecoding apps for Bloomthis Tech.

## How to use this template

1. Clone this repository:

```bash
git clone https://github.com/sciplelab/vibecode-boilerplate.git my-new-app
cd my-new-app
```

2. Open the project and start developing with your AI coding assistant.

3. Remove the existing git history from this template:

```bash
rm -rf .git
```

4. Initialize a fresh git repository for your new project:

```bash
git init
git add .
git commit -m "Initial commit from vibecode template"
```

5. (Optional) Connect to your own remote and push:

```bash
git remote add origin <your-new-repo-url>
git branch -M main
git push -u origin main
```
